# 农民配合策略

## 一句话定位
> 它属于角色策略，是用来解决「两个农民怎么分工——谁顶、谁送、谁跑——把地主的牌权和牌力一起耗光」的。

## 🎯 核心收获 · 重点知识
**一句话成果**：学完你现在能用「算顶送跑」四大核心技巧 + 上下家差异化策略 + 反向出牌法，把两个农民拧成一股绳，让地主空有牌权却打不出手。

**重点结论**（只收 AI 与你达成的最终共识，3–7 条；筛掉反复出现、一开始没做好后被推翻的）：
- 农民获胜公式 = 顶（拦路）+ 送（接力）+ 跑（冲刺）三位一体，缺一项都会被地主各个击破。
- 算牌三要素：算准地主单牌过到几 + 地主下家单牌过到几 + 地主手上没有过的小对；记 7 和 10 是顺子关键张。
- 顶牌铁律：地主上家必须顶牌（单不过 10 双不过 6），用 10 以上散牌顶，首发单张优先出 A 暴露大牌分布。
- 送牌三步骤：找准定位（牌难跑就送队友 / 队友难就送你跑）→ 配合队友 → 送对牌型（知道队友需要什么、地主没有什么）。
- 跑牌三技巧：抢先上手（比地主先出 2）+ 炸弹该拆就拆（双王牌不好时果断拆）+ 注意整体牌型（拿到炸弹后留小单 / 对）。
- 反向出牌法：地主出单你出对、地主出对你出单，打乱地主节奏；不压自己人铁律——地主不要队友的牌时不压（除非能一手跑完）。
- 信号传递三技巧：出牌要权（出 6/7 = 渴望拿牌权）+ 推磨子（拿权后出小对子试探）+ 让牌信号（辅助出大牌后出小牌）。
- 特殊牌局：地主剩 1 张 → 出非单张牌型转移牌权；地主剩 2 张 → 坚持出单逼拆牌。

## 📝 轻笔记
<details>
<summary>📝 轻笔记（点开看推演摘要与卡壳细节）</summary>

**视觉模型（五问 · 用户先讲）**：
- **对象**：地主上家、地主下家、地主、牌权、关键张（7/10/A/2/王）。
- **连接**：上家 ↔ 地主（顶牌拦截）；下家 ↔ 地主（抢牌权）；上家 ↔ 下家（送牌接力）。
- **流向**：算牌 → 顶 / 送 / 跑 分工执行 → 信号传递 → 收尾。
- **边界**：队友能一手跑完时打破「不压自己人」；地主剩 1/2 张时切换特殊牌局逻辑。
- **变化**：地主剩余张数决定配合模式（多张 = 常规配合；1 张 = 转牌权；2 张 = 逼拆）。

**视觉模型结构（AI 校准后）**：
- **chart_type（AI 自主判定，用户不输入）**：flow
- **节点 kind**：actor（上家/下家/地主）、process（算/顶/送/跑）、rule（不压自己人）、boundary（特殊牌局）
- **结构纠偏**：用户原想画成「顶→送→跑」线性流程，AI 校准为「算牌前置 + 顶送跑并列三路 + 信号贯穿」——三者不是先后而是同时由不同角色承担。

**类比**：它像「接力赛跑 + 篮球场挡拆配合」，因为顶 = 第一棒拦路（挡拆）、送 = 传递接力棒、跑 = 最后一棒冲刺；任何一棒掉链子整队就输。

**大白话讲解（讲给外行）**：你和一个队友打地主，俩人得有分工。坐在地主上家的人负责「挡」——地主出什么你都用大牌压住，别让他舒服过小牌，单张不出过 10、对子不出过 6。坐在地主下家的人负责「跑」——抢到出牌权就往出冲，能跑多快跑多快。俩人之间还要互相「送」——你知道队友缺什么牌型就出什么，地主没什么你就出什么。配合时记着：地主只剩 1 张时别出单张（出对子或三张转移牌权），地主剩 2 张时死命出单张逼他拆牌。最重要的是别和队友抢牌——地主都不要的牌你也别压，除非你能一把跑完。

**三个应用例子**：
1. 你是地主上家，地主首发单张 4。你手里有 K、A、2——按「单不过 10」铁律必须用 A 或 2 顶（用 K 不够顶），首选 A 暴露大牌分布，省下 2 给关键节点。
2. 你是地主下家，队友（上家）出小对子 33。地主不要，你手里有 44、55——按「不压自己人」铁律不要压（除非能一手跑完），让队友继续出牌。
3. 地主剩 2 张，你是上家刚拿到牌权。按「地主剩 2 张出单逼拆」规则出单张，逼地主拆对子或拆王炸——即便你出的单张很小，地主也得拆大牌压。

**失效边界**：这个概念在什么情况下不适用 / 会出错？
- 队友明确能一手跑完时，「不压自己人」失效——该让路就让路，别挡队友。
- 地主剩 1 张时还出单张 = 直接送地主赢，必须切到「出非单张转移牌权」。
- 自己手牌极差（无大牌无炸弹）时强行「跑」会拖累队友，应转入纯辅助模式只顶不跑。

**边界补丁**（失败边界 → 最小补救动作；只演练最容易翻车的 1 个）：
- 边界 1（地主剩 1 张还出单张送赢）→ 补救动作：地主剩 1 张时拿到牌权一律出对子 / 三张 / 顺子等非单张牌型，把牌权转给队友或逼地主接不上；自己手里若只有单张则优先出最大的单张（如 2 或 A）让队友判断你没有非单张。
- 边界 2（队友能一手跑完你还压）→ 补救动作：每次出牌前先看队友剩余张数，队友剩 ≤ 4 张且你判断其牌型完整时，一律 pass 让路；除非地主主动出牌且你不压就会让地主跑掉，才考虑压一次再转牌权给队友。
- 本轮演练：地主剩 1 张，你是上家拿到牌权，手里有单 5 和对 88。错误出单 5（地主直接走完），正确出对 88 转牌权——即便地主有对子能压，也消耗了地主一张大牌。

**卡壳点复盘**：学的过程中哪里卡住、哪里曾经含糊带过？补上后的正确理解是什么？
- 卡壳点：「不压自己人」一开始被理解成「绝对不压队友」，结果错过队友能跑完时的让路时机。
- 正确理解：「不压自己人」前提是「队友不能一手跑完」；队友能跑完时反而要主动让路，这才是真配合。

**成长对比**：起点认知「农民就是各打各的、谁抢到牌权谁跑」→ 现在的理解「农民是分工明确的接力赛，顶送跑 + 信号 + 特殊牌局规则缺一不可」。
