# 出牌顺序规划

## 一句话定位
> 它属于〔斗地主·出牌战术〕，是用来解决〔一手 17 张牌拆成几个轮次、按什么顺序出才能用最少手数走完且不被压死〕的——从凭感到有规划的分水岭。

## 🎯 核心收获 · 重点知识
**一句话成果**：学完你现在能在拿到牌后 3 秒内识别完整牌型，并按"拆后总手数 < 拆前 + 拆出牌型有回手机制 + 保留大牌炸弹压场"三原则规划出牌顺序，把 17 张牌控制在 5–7 手走完。

**重点结论**（只收 AI 与你达成的最终共识）：
- 拆牌三原则缺一不可：手数减少原则（拆后总手数必须 < 拆前）+ 回手牌原则（拆出的牌型要有回手机制）+ 控场优先原则（保留大牌和炸弹用于压场）。三者是 AND 关系，任一不满足就不拆。
- 5–7 手走完 17 张是拆牌成功的硬指标；8–10 手就是手数失控，必输。
- 三大经典拆法：拆对补单（拆中等对子补单牌缺口）/ 拆三补缺（拆三张补顺子缺张）/ 拆炸保控（拆小炸成四带二保大牌控场）。
- 结构保留黄金比：2 炸 + 1 顺 + 2 对 + 3 单 ≈ 17 张，是控场型牌型的理想骨架。
- 3 秒内识别完整牌型是高手门槛——拿到牌先扫断张、再看大牌分布、最后定骨架。
- 角色差异化：地主大胆主动拆长顺打节奏；农民协同拆牌 + 送牌辅助队友。
- 类比"做饭食材分拣"：先按菜式分组（拆牌），再按烹饪顺序下锅（出牌顺序）。

**重点知识大纲 / 脑图**：
- 拆牌三原则（决策门槛）
  - 手数减少原则：拆后手数 < 拆前
  - 回手牌原则：拆出牌型有回手
  - 控场优先原则：保留大牌 + 炸弹
- 三大经典拆法
  - 拆对补单（拆中等对子补单牌）
  - 拆三补缺（拆三张补顺子缺张）
  - 拆炸保控（拆小炸成四带二）
- 结构黄金比
  - 2 炸 + 1 顺 + 2 对 + 3 单 = 17 张
- 手数评估
  - 5–7 手 = 成功
  - 8–10 手 = 失控
- 角色差异
  - 地主：主动拆长顺打节奏
  - 农民：协同拆牌 + 送牌

## 📝 轻笔记

<summary>📝 轻笔记（点开看推演摘要与卡壳细节）</summary>

**视觉模型（五问 · 用户先讲）**：
- **对象**：17 张手牌、牌型组合、手数、大牌/炸弹、角色
- **连接**：牌型 → 手数；拆法 → 手数变化；角色 → 拆法选择
- **流向**：拿到牌 → 扫断张 → 定骨架 → 拆牌三原则校验 → 定出牌顺序
- **边界**：手数 ≥ 8 必拆减手；缺回手牌不拆；缺控场牌不拆
- **变化**：随出牌进程，手数动态变化，需重新规划

**视觉模型结构（AI 校准后）**：
- **chart_type（AI 自主判定）**：flow
- **节点 kind**：process（流程）/ rule（原则）/ data（手数）/ boundary（边界）/ concept（拆法）
- **结构纠偏**：用户原以为"拆牌=把对子拆成单牌"，AI 校准为"主动打破牌型结构换取灵活性与控制力"，本质是手数 + 回手 + 控场三原则联合决策。

**类比**：它像做饭前的食材分拣——先把冰箱里的食材按菜式分组（拆牌成牌型），再按烹饪先后顺序下锅（出牌顺序）；分拣错了，再好的食材也做不出一桌菜。

**大白话讲解（讲给外行）**：斗地主每人 17 张牌，不能一张一张出，得把牌分成几组（牌型）一组一组出。比如 34567 是一组顺子，555 带一张 3 是一组三带一，每组就是"一手"。怎么分才能让总手数最少、不会被对手压死、还能保住大牌压场？这就是拆牌。三原则是：①拆完手数得变少；②拆出去的那组得有"回手机会"（自己能再接回出牌权）；③得留大牌和炸弹用来最后压场。5–7 手走完 17 张就是高手，8–10 手就是新手乱拆。

**三个应用例子**：
1. **拆对补单**：手牌 KK + 散牌 3、5、7、9。KK 拆成两张 K 单出，能压对手单牌拿回出牌权，比留着 KK 被对手 AA 压死强。
2. **拆三补缺**：手牌 777 + 34568。777 拆掉，3+4+5+6+7+7+7 组成 34567 顺子 + 77 对子，手数减少且顺子有回手机会。
3. **拆炸保控**：手牌 3333 + 大王 + 2。3333 拆成四带二（3333 带大王 + 2），保留单张控制权，避免后期没大牌。

**失效边界**：对手有王炸时拆牌压不住（必须先诱出王炸再拆）；自己单牌过多（≥ 5 张）且无大牌时拆牌无意义（直接送小单等队友）；残局阶段（每人 ≤ 4 张）拆牌三原则失效，按残局算法走。

**边界补丁**（失败边界 → 最小补救动作；只演练最容易翻车的 1 个）：
- 边界 1（对手有王炸）→ 补救动作：先连续出单牌逼对手拆王，再启动拆牌规划。
- 边界 2（单牌过多无大牌）→ 补救动作：放弃拆牌规划，改为"送小牌 + 留一张大牌防偷跑"。
- 本轮演练：边界 1——模拟地主出小王逼农民出大王，再启动拆对补单。

**卡壳点复盘**：一开始以为"拆牌 = 减少手数"，忽略回手牌原则——拆出顺子后被对手炸弹压死，没法回手。补上后理解：拆牌三原则是 AND 关系，不是 OR。

**成长对比**：起点认知"拆牌就是把对子拆成单牌出" → 现在的理解"拆牌是手数 + 回手 + 控场三原则联合决策，目标是 5–7 手走完 17 张且保留控场能力"。
