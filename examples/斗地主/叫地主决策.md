# 叫地主决策

## 一句话定位
> 它属于叫牌决策，是用来解决「拿到这手牌到底叫不叫、叫几分」的——叫牌是斗地主一切胜负的起点，叫错一盘可能要三盘才能补回来。

## 🎯 核心收获 · 重点知识
**一句话成果**：学完你能用「牌力基础分 + 位置修正分 + 底牌期望分」三维决策模型，在 10 秒内对任意手牌得出"叫 3 / 叫 1 / 弃叫"的明确结论，并把"叫错代价 > 不叫损失"内化为 70% 胜率的核心护城河。

**重点结论**（AI 与你的最终共识）：
- **叫牌 = 三维决策，不是单维牌力**：牌力基础分（点力）+ 位置修正分（叫牌顺序）+ 底牌期望分（补强概率）——三维加权后才能定档。只看牌力不看位置和底牌期望，等于盲叫。
- **牌力三档划分**：≥7 分强牌区（叫 3 / 2 分）/ 5-6 分灰色地带（看位置与底牌）/ ≤4 分弱牌区（直接弃）。强牌区还要校验"断张≤2 + 手数≤5 + 有控制力"才能叫 3 分。
- **位置策略差异巨大**：首家保守（阈值最高，叫 3 分需点力 ≥8 + 结构完整）/ 二家可捡漏可拦截（看首家叫分判断其牌力）/ 三家末家阈值最低（前两家弃叫表明牌力弱，底牌补强概率 >60%，可下调 10% 阈值反叫）。
- **底牌期望概率**：断 1 张补到概率 ~42% / 断 2 张 ~27% / 断 3 张 ~12%——断 3 张以上别赌底牌补顺，基本是赌输。
- **"叫错代价 > 不叫损失"是 70% 胜率核心护城河**：叫错当地主失败 = 输 2 倍底分给两个农民 + 翻倍风险；不叫仅失去底牌补强机会，仍可当农民赢。**宁可弃叫也别错叫**——这是新号保胜率的第一铁律。
- **抢地主制 vs 叫分制策略差异**：抢地主制下，已有叫分基础再抢会翻倍，需更高牌力门槛（≥8 分才考虑抢）；叫分制下则按 1→2→3 分递增决策。
- **3 个实战案例验证三维模型**（见轻笔记）。

**重点知识大纲 / 脑图**：
- 三维叫牌决策模型
  - ① 牌力基础分（点力）
    - ≥7 分强牌区 → 倾向叫 3 / 2 分
    - 5-6 分灰色地带 → 看位置与底牌
    - ≤4 分弱牌区 → 直接弃
  - ② 位置修正分
    - 首家：阈值最高，保守（≥8 分才叫 3 分）
    - 二家：可捡漏可拦截（看首家叫分）
    - 三家：阈值最低，可下调 10%（前家弃叫=底牌补强概率 >60%）
  - ③ 底牌期望分
    - 断 1 张补到概率 ~42%
    - 断 2 张 ~27%
    - 断 3 张 ~12%（基本赌输）
- 决策输出三档
  - 叫 3 分：强牌 + 断张≤2 + 手数≤5 + 有控
  - 叫 1 分：灰色地带 + 末家 + 断 1 张
  - 弃叫：弱牌 / 炸弹陷阱 / 首家结构断裂
- 核心护城河
  - 「叫错代价 > 不叫损失」
  - 新号保 70% 胜率第一铁律：宁可弃叫也别错叫
- 机制差异
  - 叫分制：1→2→3 分递增
  - 抢地主制：抢后翻倍，门槛更高（≥8 分才抢）

## 📝 轻笔记
<details>
<summary>📝 轻笔记（点开看推演摘要与卡壳细节）</summary>

**起点认知**：「叫地主就是看手牌好不好，感觉好就叫 3 分」——单维决策，凭感觉。

**视觉模型（五问 · 用户先讲）**：
- **对象**：手牌点力、叫牌位置、底牌补强概率、三家叫分顺序
- **连接**：点力↔牌力档位；位置↔阈值修正；底牌↔补强概率
- **流向**：发牌 → 牌力分档 → 位置修正 → 底牌期望 → 输出叫分（3/1/弃）
- **边界**：炸弹陷阱、首家结构断裂、断 3 张赌底牌
- **变化**：底牌翻开后可能改变结构（补断/升级炸弹）——但决策在叫牌时已定

**视觉模型结构（AI 校准后）**：
- **chart_type**：flow（发牌→牌力分档→位置/底牌修正→叫分输出→护城河反馈，主流程清晰）
- **节点 kind**：process(发牌)、data(牌力分)、boundary(三档区)、rule(位置/底牌修正)、state(叫分输出)、concept(三维合并)、feedback(护城河)
- **结构纠偏**：用户初版想用 hierarchy（层级），AI 校准为 flow——叫牌是时序决策流程（发牌→评估→修正→输出），非并列层级；末尾用 feedback 节点回指"叫错代价>不叫损失"的护城河循环。

**类比**：它像**德州扑克的 preflop 决策**——手牌强度（点力）+ 位置（button/cutoff/UTG）+ 赔率期望（底池胜率）三维加权，专业玩家从不只看手牌。斗地主叫牌同理：牌力是基础，位置和底牌期望是修正项，三维合并才出决策。

**大白话讲解（讲给外行）**：叫地主不是"感觉大牌就叫"这么简单，要看三件事：第一，**牌力基础分**——按点力算（大王 4、小王 3、2 每张 2、A 每张 1），7 分以上是强牌，5-6 分是灰色地带，4 分以下直接弃；第二，**位置**——你是第几个叫牌的很重要，首家最保守（不知道后面情况），三家末家最激进（前面两家弃叫说明他们牌差，底牌补强概率高）；第三，**底牌期望**——你断几张牌决定了底牌能不能补上，断 1 张补到概率 42%，断 3 张只剩 12% 基本赌输。三维合并才能定档叫几分。最关键的一条铁律：**叫错的代价远大于不叫的损失**——叫错当地主输了要付双倍底分给两个农民，不叫只是失去补强机会还能当农民赢。所以新号保 70% 胜率，宁可弃叫也别错叫。

**三个应用例子**：
1. **强牌叫 3 分**：首家拿大王+小王+2 张 2+对 A+5 顺子，点力 11 分、断张 1、有控、手数 4 → 强牌区+结构完整 → 直接叫 3 分
2. **灰色地带弃叫**：首家拿小王+1 张 2+对 K+散牌 5 张，点力 5 分、断张 3、无大王控制 → 灰色地带但首家保守阈值+结构断裂 → 弃叫
3. **末家反叫 1 分**：前两家弃叫，三家拿 1 张 2+对 A+对 K+5 顺子，点力 6 分、断 1 张 → 灰色地带+末家+底牌补强概率高 → 反叫 1 分（前家弃叫=底牌补强概率 >60%）

**失效边界 / 不适用场景**：
- **底牌爆冷**：断 1 张（概率 42% 补到）仍可能补不到，叫牌后底牌翻出三张废牌 → 决策正确但运气不利
- **对手埋伏**：农民故意弃叫埋伏，实际牌力很强 → 叫了地主才发现被埋伏
- **抢地主制高倍场**：抢后翻倍风险陡增，常规牌力阈值不适用，需更高门槛（≥8 分才考虑抢）
- **高倍场保守原则**：底分已高的场次，叫错代价倍数放大，应进一步上调叫牌阈值

**边界补丁**（失败场景 → 补救动作）：
- **底牌爆冷**（断 1 张没补到，反而补三张废牌）→ 补救：出牌阶段改为防守型，先出绝对控制牌稳住牌权，把损失控制在不被春天的范围
- **对手埋伏**（叫了 3 分发现农民有王炸）→ 补救：记牌算王炸位置，避免在关键回合被双倍收割，争取先出完
- **抢地主制抢后翻倍**（抢了才发现牌力不够）→ 补救：出牌优先打乱农民配合，用炸弹换牌权速战速决，避免拖延到被翻倍结算
- **本轮演练**：首家拿小王+2 张 2+对 A+对 K+散牌 5 张，点力 7 分、断 2 张 → 强牌区但首家保守+结构临界 → 按三维模型应叫 1 分而非 3 分（首家+断 2 张需降档），验证"首家保守阈值"

**卡壳点复盘**：
- **核心卡壳**：「为什么叫错代价大于不叫损失？」→ 补：叫错当地主失败 = 输 2 倍底分给两个农民 + 翻倍风险（炸弹/春天叠加），总损失可达 4-8 倍底分；不叫仅失去底牌补强机会，仍可当农民赢牌，损失上限仅 1 倍底分。**期望值（EV）上，错叫的负 EV 是弃叫的 3-5 倍**。
- **次要盲区**：最初以为位置只是顺序不影响决策 → 补：末家阈值可比首家低 10%，因为前两家弃叫是强信息（牌力弱+底牌补强概率 >60%）。
- **三重验证**：① 强牌例（点力 11+断 1+有控）→ 叫 3 分 ✓ ② 灰色弃叫例（点力 5+断 3+首家）→ 弃叫 ✓ ③ 末家反叫例（点力 6+断 1+末家）→ 叫 1 分 ✓，三维模型一致

**成长对比**：起点「叫地主看手牌好坏，感觉好就叫 3 分」→ 现在「三维决策（牌力+位置+底牌期望），叫错代价>不叫损失是 70% 胜率护城河，宁可弃叫也别错叫」。

### 1. 徐晓轶 · 「剩余牌手数理论提出者」
- **身份**：IT 工程师、知乎专栏作者，用蚁群算法研究斗地主牌型整理
- **与本概念的关系**：他的"剩余牌手数"理论为叫牌决策提供了控制力维度的量化方法——叫牌前评估的不只是点力，还有"出小牌后能否不耗炸续控"，这正是三维决策中牌力基础分背后的隐藏维度
- **核心洞见**：「斗地主的关键就是争夺牌权；同类型累加后的剩余牌手数反映了打了一手小牌之后能不能收回来，即不需要额外消耗炸弹而继续保有出牌权。」——这一洞见解释了为什么"有炸弹但缺 2 缺王"的炸弹陷阱必须弃叫：剩余牌手数累加为正，意味着要消耗自己的炸弹才能出牌，控制力为负。（原话引述自知乎专栏）
- **代表作**：知乎专栏《斗地主之用蚁群算法整理牌型》系列（2017）
