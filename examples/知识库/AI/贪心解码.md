# 贪心解码

## 一句话定位
> 它属于 AI 大模型的**解码策略**，规则极简——每一步都选**当前概率最高**的那个 token。

## 🎯 核心收获 · 重点知识
**一句话成果**：你现在能解释 `temperature=0` 为什么结果稳定可复现，以及它什么时候不够用。

**重点结论**
- 模型每步输出的是**整个词表上的概率分布**；贪心 = 每步永远取 **argmax**（最大那个）。
- **确定性**：同样输入 → 永远同样输出（无随机性）。
- **代价**：每步局部最优 ≠ 全局最优，容易陷入**重复、平庸**的句子（短视）。
- **对比**：温度采样 / top-p / beam search 都是为缓解贪心的"短视"而生。

**例子**：输入"中国的首都是"，每步选最高概率 token → "北京"。
**换场景举例**：分类/信息抽取要**稳定可复现**时，用贪心（temperature=0）保证每次结果一致。
